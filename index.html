What I Do

Vue.js

I love this framework most. I have built several projects for my clients, an Open Source project and it is my first choice, so I'm using it also for my private projects as well.

Node.js

I like the idea of one programming language for both, the FE and BE. It is a good choice for building lightweight but scalable APIs. My choice No 1 for backend recently.

MongoDB

Although I was widely using SQL databases, I love the simplicity of storing JSON and handling it in javascript. When it is appropriate I always choose MongoDB as my data storage.

.NET Core & Ruby (on Rails)

I'm a frontend developer yet I have more than 11 years of experience in BE development. When I need to build an API for my projects, Node.js, Ruby and .NET Core are my choices.
